Colophon Made by Chris Wan. Designed in Sketch, Illustrator, and Photoshop. Handcoded using HTML, Sass-extended CSS, and JavaScript in Atom. Parsed by Jekyll. Hosted by GitHub Pages. Images compressed by psychovisual image encoder Guetzli. Akzidenz Grotesk and Adobe Garamond used for type. Responsive layout has a fluid 1248px-wide tripartite grid divisible into two 624px-wide columns, three 416px-wide columns, four 312px-wide columns, six 208px-wide columns, and so on, with vertical rhythm of 30px. Designed and coded to be readable on desktop, tablet, and mobile.
